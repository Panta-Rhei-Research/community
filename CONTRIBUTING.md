# Contributing

Thank you for considering a contribution.

Panta Rhei is an open independent research program. Contributions can improve the public research object without implying endorsement of the framework.

## Contribution types

You can contribute by:

- fixing broken links or metadata;
- improving documentation;
- improving accessibility;
- improving search/indexing;
- improving page templates;
- improving publication artifacts;
- improving import reports;
- improving TauLib documentation;
- improving Lean formalization where appropriate;
- reviewing result status or source mappings;
- proposing clearer wording.

## Before contributing

For conceptual questions, start in Discussions.

For concrete defects, open an Issue.

For concrete changes, open a Pull Request.

## Repository routing

- `site`: website source, templates, navigation, generated pages.
- `publications`: publication artifacts, PDFs, notes, briefings, supplements.
- `taulib`: Lean formalization.
- `research`: scripts, notebooks, import reports, exploratory supplements.
- `community`: discussion guidance, templates, onboarding, governance.

## Pull Request expectations

A good PR should:

- explain what it changes;
- link relevant issue/discussion if one exists;
- keep claims status-safe;
- avoid changing scientific meaning without discussion;
- pass local build checks where applicable;
- preserve citations and source metadata.

## Non-endorsement

Contributing a fix does not imply acceptance of the theory.
