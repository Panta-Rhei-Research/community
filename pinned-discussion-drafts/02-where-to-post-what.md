# Where to Post What

Use this routing rule:

- Public questions and critique: GitHub Organization Discussions.
- Concrete defects: Issues in the relevant repository.
- Concrete edits: Pull Requests.
- Private, institutional, media, sensitive, or support-related contact: email.
