# Welcome - Start Here

Welcome to the public discussion space of the Panta Rhei Research Program.

The website is the public research observatory. GitHub is the public engagement, source, formalization, contribution, and correction substrate.

Start here if you are not sure where to enter:

- Website: https://panta-rhei.site
- How to engage: https://panta-rhei.site/engage/
- Public discussions: https://github.com/orgs/Panta-Rhei-Research/discussions
- Community hub: https://github.com/Panta-Rhei-Research/community

Participation does not imply endorsement. You can ask, inspect, challenge, review, correct, or contribute without accepting the framework.
