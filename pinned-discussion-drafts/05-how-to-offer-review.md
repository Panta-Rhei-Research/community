# How to Offer Review

The program is large, so the most useful review is bounded.

Please include:

- area of expertise;
- target page or artifact;
- review mode;
- whether review can be public;
- available scope;
- any conflict or affiliation disclosure.

Useful targets include one Problem Ledger item, one Recovery Requirement, one Construction Spine step, one Result page, one Research Note, one TauLib module, one Numerical Physics Ledger entry, or one website lane/template.
