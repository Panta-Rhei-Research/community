# 🧭 Where to discuss what

Use this quick routing guide to choose the right place.

## 💬 community
Use for:
- broad framework questions
- project-wide ideas
- onboarding and orientation
- discussions that do not yet belong elsewhere

## 🧮 taulib
Use for:
- Lean / TauLib implementation
- formal kernel questions
- proof and module discussions
- code pull requests

## 🧪 research
Use for:
- paper sources
- notebooks
- scripts
- exploratory technical materials
- topic-specific research collaboration

## 📚 books / 🧭 topics / ✨ results / 📄 papers / 🧮 formalization / 🚀 project
These are primarily **read-first official surfaces**.

Use them mainly for reading and reference.  
Bring general discussion back to `community` unless a maintainer explicitly asks otherwise.
